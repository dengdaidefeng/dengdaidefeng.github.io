java链接Redis常用框架

  • jedis 上手简单，整体和redis本身操作差异不大
  • lettuce
  • Redisson
  • SpringDataRedis 包含了上面两种框架，将链接redis解耦，可写在配置文件中，使用方便

jedis

注：jedis本身是线程不安全的，在高并发的情况下很可能出现线程安全问题

  1. 使用jedis本身连接redis
  1. 使用Jedis连接池连接redis

SpringDataRedis

  • 对客户端jedis和lettuce的整合
  • 提供RedisTemplate统一API来操作Redis
  • 支持Redis的发布订阅模型
  • 支持Redis哨兵和集群
  • 支持基于lettuce的响应式编程
  • 支持基于JDK，Json，字符串，String对象的数据序列化和反序列化
  • 支持基于Redis的JDKCollection实现

    使用StringRedisTemplate创建对象，它的key，value序列化方式默认就是String方式，不需要我们自定义RedisTemplate（自定义的话数据中会写入这个数据所在的类）
